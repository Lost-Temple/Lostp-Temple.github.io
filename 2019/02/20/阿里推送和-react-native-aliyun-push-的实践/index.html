阿里推送和 react-native-aliyun-push 的实践

发表于 2019-02-20 | 分类于 react-native 花里胡哨的实践

这次做 app 的推送功能，用的是阿里云的推动。github 上找到了 react-native-aliyun-push 这个组件，看了一下功能都能满足。根据文档，很快就完成了接入。基本没有什么问题。需要注意的大概有三点。

  1. iOS 上 deviceToken 注册失败，返回

这是 Xcode 里面有一个开关没打开造成的，如图

  1. iOS 上如果应用同时继承了支付宝支付，可能会报 UTDID 冲突，可以去支付宝平台 下载新的 SDK ，如图

  2. Android8.0 以上设备需要配置 notificationChannel ，前端和后台要配置相同的 notificationChannel，阿里云移动推送的官方文档有提到，但是容易被忽略。如果发现 Android 设备收到消息有数据打印，但是没有弹出通知，有可能就是没有配置 notificatinoChannel ，也可能是后台推送的 type 是 message 而不是 notification。
